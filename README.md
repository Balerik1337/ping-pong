# ping-pong
-----НАЗВАНИЕ ИГРЫ-----
Это игра пинг-понг!


----------ОПИСАНИЕ ИГРЫ----------
Игра является мультиплеерной, и управляется с помошью клавиш ВВЕРХ ВНИЗ (Правый игрок) и W S (Левый игрок).
